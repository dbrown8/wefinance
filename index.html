Advance Enquiry Form

This is an enhanced version of the inquiry form, which incorporates the following features:

Dynamic Rules:

Dynamic rules allow for conditional display of form fields and automated calculation of values based on user inputs, enhancing form interactivity and functionality. For more details.

  • Visibility Expression: The Email field dynamically appears when users select the subscribe checkbox.
  • Value Expression: The Estimated amount is automatically calculated based on the specified number of days (start and end date) and room budget.

Repeatable Panel:

  • Traveler Information: Users can conveniently add multiple traveler details, with flexibility for a minimum of 1 and a maximum of 3 travelers per submission. More details in repeatable tutorial

Integrations:

Google reCaptcha:

  • Implemented to enhance security measures, effectively preventing fraudulent activities, spam, and abuse. For integrating it into your own form, refer to the provided tutorial.
